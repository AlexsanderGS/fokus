# 🎯 Fokus

Projeto desenvolvido durante o curso **JavaScript: manipulando elementos no DOM**, da Alura.

O objetivo deste projeto foi praticar **manipulação do DOM utilizando JavaScript**, explorando como alterar elementos da interface de forma dinâmica, reagindo a interações do usuário.

---

## 📚 Sobre o Projeto

O **Fokus** é uma aplicação inspirada na técnica **Pomodoro**, utilizada para ajudar na concentração durante períodos de estudo ou trabalho.

A aplicação permite alternar entre diferentes modos de foco:

- 🧠 Foco
- ☕ Pausa curta
- 🛌 Pausa longa

Durante o uso, o JavaScript manipula diversos elementos da página para:

- Alterar textos
- Trocar imagens
- Modificar classes CSS
- Controlar o tempo do cronômetro
- Responder a eventos do usuário

Tudo isso utilizando **JavaScript puro (Vanilla JS)** e **manipulação direta do DOM**.

---

## 🧠 Conceitos praticados

Durante o desenvolvimento deste projeto foram trabalhados conceitos importantes de JavaScript, como:

- Seleção de elementos do DOM
- Manipulação de conteúdo com `textContent`
- Alteração de classes com `classList`
- Manipulação de atributos
- Eventos (`addEventListener`)
- Funções e organização de código
- Controle de tempo com `setInterval`
- Lógica de interação com o usuário

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Manipulação do DOM
